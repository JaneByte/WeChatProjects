# 小程序演示回归清单

## P0 必测链路

1. 登录初始化：进入首页后可正常获取 userId，个人中心显示当前用户。
2. 商品浏览：首页/分类页可正常加载商品，商品详情可打开。
3. 加入购物车：分类页和商品详情页加购成功，购物车可看到对应商品。
4. 购物车操作：数量加减、单选/全选、删除、批量删除正常。
5. 结算下单：选择地址后可提交订单，跳转到支付结果页。
6. 支付流程：支付结果页可进入订单列表，订单状态从待付款变为待发货。
7. 订单流转：待发货可取消，待收货可确认收货，订单详情和订单列表状态一致。

## 异常场景

1. 无网络时：页面能提示“加载失败/请求失败”，不会白屏。
2. 空数据时：购物车空态、订单空态、地址空态显示正常。
3. 库存不足：加购/下单时有明确提示，不会提交成功。

## 演示前检查

1. 前端中文无乱码（重点看首页、分类、购物车、订单页）。
2. 后端服务已启动，数据库连接正常。
3. 使用同一 userId 演示，避免数据混乱。
